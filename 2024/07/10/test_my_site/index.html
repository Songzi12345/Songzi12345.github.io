Hello world

提前安装好(node.js)[https://nodejs.org/en] 左边的那个

提前安装好(GIt)[https://git-scm.com/downloads]

两软件安装路径不能为中文
提前注册好一个github账号

在本地找个目录放博客源文件，路径不要有中文（比如我在C:创建一个放博客源文件blog文件）并在此路径下打开Git Bash

打开界面后使用命令安装hexo
npm install -g hexo-cli //安装hexo
hexo init blog //初始化一个名为blog的项目
cd blog //进入blog
hexo g //编译构

安装完之后发现blog文件下有多目录（蓝色）和文件（白色）


使用命令在blog目录下创建一个新的文件test_my_site,并发现其安装路径是C:\blog\source_posts


使用命令创建并启动服务
hexo g //构建
hexo s //启动服务
之后就可以访问127.0.0.1:4000访问blog了（也可以Ctrl+鼠标左击http:localhost:4000/）

访问界面，可以使用Ctrl+c退出服务

退出服务，配置ssh密钥（先查看本地是否已存在ssh密钥）
使用密令ssh-keygen -t rsa -C “GitHub邮件地址”，然后连续3次回车

生成一个文件在C:\Users\Administrator目录下的.ssh文件，打开.ssh文件找到.ssh\id_rsa.pub文件，记事本打开并复制里面的内容（相当于密钥）

打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key

使用命令ssh -T git@github.com尝试ssh连接,测试是否成功（出现Hi，。。。表示测试成功）
